Flowers for Ingersoll and Freed

Where: Kaladi Brother's Coffee

When: 2/17, 2/24, 3/3, 3/10

Time: 3:00PM - 5:00PM

The Arlene's Flowers lawsuit is a group of merged civil suits brought against Arlene's Flowers of Richland, Washington, by a same-sex couple, Robert Ingersoll and Curt Freed, that was refused service. The florist refused to sell them flowers for their wedding because of their sexual orientation. The suit says that the refusal of Arleneâ€™s Flowers to sell flowers to the couple violates the Washington Law Against Discrimination. The case touches upon a larger question of whether business owners should have the right to refuse service based on religious beliefs. Read more about it here.

During this module, we will be learning basic programming concepts in order to build flowers for Ingersoll and Freed, the couple denied service at the flower shop, display it on a webpage, and publish it to the internet. We will be using the programming language TypeScript and the open source library p5.js.

The awesome spinning flower above was built by Dane!

Topics Covered:

  • Basic Syntax in TypeScript and p5.js
  • Data types
  • Functions
  • Iteration
  • Git
  • Github Pages

Prerequisites:

Although no coding knowledge is required for this module, we do ask that our mentees having a basic understanding of how to use a computer (run a program, download and install software).

Until we have the resources to provide computers for our mentorship, our mentees are required to have machines that can run code editors. This is usually a Windows Laptop/Macbook. Certain tablets will work too.

We also ask that mentees install the following before coming in for the first session:

  • Visual Studio Code
  • Git
  • npm
  • Sign up for a Github Account

Please shoot Alessandro an e-mail if you have issues installing any of this software!
